# Timer Xadrez do Povo (PWA)

Um timer de xadrez simples, bonito e responsivo, feito em HTML, CSS e JavaScript, pronto para ser instalado como aplicativo PWA no seu celular.

## Como usar
1. Coloque o arquivo `logo.jpeg` na pasta `pwa/`.
2. Abra o arquivo `index.html` no navegador.
3. Para instalar como app, use a opção "Adicionar à tela inicial" do navegador.

## Funcionalidades
- Timer de 5 minutos com botão iniciar/parar.
- Design moderno e responsivo.
- Funciona offline (PWA).
- Usa o logo personalizado.

## Estrutura
- `index.html`: Página principal.
- `style.css`: Estilos visuais.
- `app.js`: Lógica do timer.
- `manifest.json`: Configuração PWA.
- `service-worker.js`: Cache offline.
- `logo.jpeg`: Logo do app (adicione aqui).
